## 核心思想：拆分变量

对于任意实数 $$x_i$$（可正可负），引入两个**非负**辅助变量：

- $$x_i' \geq 0$$ — 表示 $$x_i$$ 的**正部分**
- $$x_i'' \geq 0$$ — 表示 $$x_i$$ 的**负部分**（取绝对值后）

满足两个等式：

$$
\boxed{x_i = x_i' - x_i''} \qquad \boxed{|x_i| = x_i' + x_i''}
$$

## 为什么成立？

| 情况 | $$x_i'$$ | $$x_i''$$ | $$x_i = x_i' - x_i''$$ | $$x_i' + x_i'' = \|x_i\|$$ |
|------|:------:|:------:|:------:|:------:|
| $$x_i \geq 0$$ | $$x_i$$ | $$0$$ | $$x_i - 0 = x_i$$ ✓ | $$x_i + 0 = \|x_i\|$$ ✓ |
| $$x_i < 0$$ | $$0$$ | $$-x_i$$ | $$0 - (-x_i) = x_i$$ ✓ | $$0 + (-x_i) = \|x_i\|$$ ✓ |

## 谁为 0？——由 $$x_i$$ 的正负决定

$$
\begin{cases}
x_i \geq 0 \text{ 时：} & x_i' = x_i, & x_i'' = 0 \\
x_i < 0 \text{ 时：} & x_i' = 0,    & x_i'' = -x_i
\end{cases}
$$

**一句话：正数归 $$x_i'$$，负数归 $$x_i''$$（取反），另一个补 0。**

举例：

| $$x_i$$ | $$x_i'$$ | $$x_i''$$ | 验证 $$x_i = x_i' - x_i''$$ | 验证 $$\|x_i\| = x_i' + x_i''$$ |
|:-----:|:------:|:------:|:---:|:---:|
| $$5$$ | $$5$$ | $$0$$ | $$5 - 0 = 5$$ ✓ | $$5 + 0 = 5$$ ✓ |
| $$-3$$ | $$0$$ | $$3$$ | $$0 - 3 = -3$$ ✓ | $$0 + 3 = 3$$ ✓ |
| $$0$$ | $$0$$ | $$0$$ | $$0 - 0 = 0$$ ✓ | $$0 + 0 = 0$$ ✓ |

## 优化层面：最优解自动保证不同时 > 0（反证法）

转换后的模型里**没有**显式写出 $$x_i' \cdot x_i'' = 0$$ 这个约束。为什么能保证？

**证明：** 假设最优解中某个 $$x_i' > 0$$ 且 $$x_i'' > 0$$，令 $$m = \min(x_i', x_i'') > 0$$。

构造新解：
- $$x_i' \leftarrow x_i' - m$$
- $$x_i'' \leftarrow x_i'' - m$$

则：
- $$x_i = (x_i' - m) - (x_i'' - m) = x_i' - x_i''$$ **不变** ✓ （约束不受影响）
- 目标函数减少 $$2m$$：$$(x_i' - m) + (x_i'' - m) = x_i' + x_i'' - \mathbf{2m}$$

→ 与原假设"最优解"矛盾！因此最优解中 $$x_i'$$ 和 $$x_i''$$ **必有一个为 0**。

> ⚠️ **注意**：这个自动归零的性质只在目标函数是 **$$\min$$ 且系数为正**时才成立。如果是 $$\max$$，则必须额外添加约束 $$x_i' \cdot x_i'' = 0$$（或等价的大 M 约束）。

## 例题（中山大学 2017 年考研真题）

### 原问题

$$
\begin{cases}
\min \ z = |x_1| + |x_2| + |x_3| \\
\text{s.t.} \ \begin{cases}
x_1 + x_2 \leq 1 \\
2x_2 + x_3 \geq 3
\end{cases}
\end{cases}
$$

### 转化步骤

1. 令 $$x_i = x_i' - x_i''$$，$$x_i', x_i'' \geq 0$$（$$i = 1,2,3$$）
2. $$\|x_i\| = x_i' + x_i''$$
3. 代入原问题

### 转化后（线性规划）

$$
\begin{cases}
\min \ z = x_1' + x_1'' + x_2' + x_2'' + x_3' + x_3'' \\
\text{s.t.} \ \begin{cases}
x_1' - x_1'' + x_2' - x_2'' \leq 1 \\
2x_2' - 2x_2'' + x_3' - x_3'' \geq 3 \\
x_i', x_i'' \geq 0 \quad (i = 1,2,3)
\end{cases}
\end{cases}
$$
