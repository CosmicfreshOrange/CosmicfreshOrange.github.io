⚖️ 1. 物权与债权的区别 ★★★★★

对比维度物权债权
权利性质支配权请求权
效力范围对世权 / 绝对权对人权 / 相对权
排他性✅ 有排他性（一物一权）❌ 平等性
公示性✅ 登记或交付公示❌ 非公开性
法定 vs 自由物权法定原则意思自治原则
客体以物为主以行为为客体
存续期限永久性 / 无期限有特定期限
📌 口诀记忆
物权 = 支配 + 对世 + 排他 + 法定｜债权 = 请求 + 对人 + 平等 + 自治

📐 2. 物权法三大基本原则 ★★★★★

原则核心内容法条
① 平等保护原则国家、集体、私人物权受法律平等保护第3条
② 物权法定原则物权的种类和内容由法律规定，禁止当事人自由创设第116条
③ 公示公信原则公示：不动产登记 / 动产交付；公信：信赖公示内容受法律保护第208-209条
⚠️ 特别注意
违反物权法定原则 → 区分"种类法定"和"内容法定"分别确定法律后果。物权法定包括：种类法定 + 内容法定 + 效力法定

🔄 3. 物权变动模式 ★★★★★

类型变动要件法条
不动产物权变动🏠 登记生效主义——不登记，物权不变动第209条
普通动产物权变动📦 交付生效主义——交付即发生物权变动第224条
特殊动产物权变动
(船舶、航空器、机动车)		🚗 登记对抗主义——交付即变动，但不登记不得对抗善意第三人第225条
⚠️ 非基于法律行为的物权变动
继承、法院判决、征收、合法建造/拆除——事实发生时即取得物权，但再处分时必须先登记！
📝 例题1：郑某继承房屋 + 抵押案
郑某因公司资金困难，其父将3年前25万元购买的房屋卖给吴某，但未来得及登记即因车祸去世。郑某继承房屋并办理登记后，以该房屋抵押给关某借款10万元，但未办理抵押登记。
A. 郑某父亲与吴某的房屋买卖合同有效B. 郑某享有房屋所有权C. 郑某父亲去世后有义务协助吴某办过户D. 关某对房屋不享有抵押权
✅ 答案：ABCD 全选
解析：A ✓ 合同效力与登记无关（区分原则）；B ✓ 郑某通过继承取得所有权；C ✓ 继承人应继续履行被继承人合同义务；D ✓ 不动产抵押权自登记时设立，未登记=无抵押权。
📝 例题2：特殊动产——赵某购车案
赵某向姚某购买轿车一辆，付清全款并已交付，但未办理过户登记。后姚某因欠款被诉，法院依申请查封了仍登记在姚某名下的该车。赵某提出异议。
问：赵某能否对抗法院的查封？
✅ 不能对抗
解析：机动车为特殊动产，采取登记对抗主义。虽已交付取得所有权，但未办理过户登记，不得对抗善意第三人（包括法院查封）。物权区分原则（第215条）：合同有效 ≠ 物权已转移至可对抗第三人的程度。

🛡️ 4. 善意取得制度 ★★★★★

📜《民法典》第311条——五个构成要件
① 让与人无处分权｜② 受让人受让时为善意（不知情）｜③ 以合理的价格有偿转让｜④ 已完成公示（不动产登记/动产交付）｜⑤ 转让合同有效
❌ 不适用善意取得
遗失物（2年内可追回）、所有权保留买卖未付清全款、无偿赠与、盗赃物
📝 例题3：夫妻共有房屋被一方出售
甲、乙婚后购得房屋一套，仅以甲的名义登记。后甲、乙感情不和，甲擅自将该房屋以时价出售给不知情的丙，并办理了转移登记。
A. 买卖合同有效，房屋所有权未转移B. 买卖合同有效，房屋所有权已转移C. 买卖合同有效，房屋所有权已转移D. 买卖合同无效，房屋所有权未转移
✅ 答案：C（买卖合同有效，所有权已转移）
解析：甲擅自处分夫妻共有财产属无权处分，但丙为善意第三人+支付合理价格+已完成登记→构成善意取得。合同效力与物权变动需区分判断。
📝 例题4：遗失物归属——手表遗失案
甲不慎遗失手表→乙拾得后交公安机关→公告期满无人认领→交寄售店出售→丁自寄售店购买→赠与女友戊→戊在公园被小偷盗走→小偷低价卖给乙→乙遇见工人丙，将手表送给丙。
问：手表最终应归谁所有？ A.甲 B.丁 C.戊 D.丙
✅ 答案：C（戊所有）
解析：甲公告期满未认领→丧失所有权。丁从寄售店（有经营资格的经营者）购买→合法取得所有权。丁赠与戊→戊取得所有权。小偷不能取得所有权，乙从非正规渠道购得也不能取得，乙赠与丙无效。

👥 5. 按份共有 vs 共同共有 ★★★★★

对比项按份共有共同共有
份额✅ 明确份额❌ 不分份额
基础关系约定共同关系（夫妻/家庭/遗产未分割前）
处分共有物占份额 ≥2/3 同意全体共有人一致同意
份额转让其他共有人有优先购买权共有基础丧失前无优先购买权
随时分割✅ 可随时请求分割❌ 共有基础丧失或重大理由时方可
推定规则无约定+无家庭关系→推定为按份共有有家庭关系→推定为共同共有
⏰ 优先购买权行使期限
通知载明期间→从通知（≥15日）；未载明→从知悉之日起15日；无法确定知悉时间→权属转移之日起6个月

🔐 6. 担保物权三大类型对比 ★★★★

对比项抵押权质权留置权
标的物不动产 / 动产动产 / 权利动产
是否转移占有❌ 不转移✅ 转移已合法占有
设立方式约定约定⚡ 法定（无需约定）
不动产设立登记生效——
动产设立登记对抗交付生效法定条件满足时设立

📌 口诀
不动产抵、动产质、法定留置——抵押权看登记、质权看交付、留置权看法定条件

🏢 7. 建筑物区分所有权 ★★★★

三部分构成
① 专有权（对专有部分独占）② 共有权（对共有部分按份共有）③ 共同管理权/成员权
业主大会议题参与表决门槛通过门槛
一般事项（制定规约、选聘物业等）面积+人数 ≥ 2/3参与表决的过半数
重大事项（筹集维修资金、改建重建、改变共有部分用途等）面积+人数 ≥ 2/3参与表决的 ≥ 3/4
📝 例题5：住宅楼共有部分权利判断
甲、乙、丙、丁分别购买某住宅楼一至四层。判断下列说法的正误：
① 四人可分享外墙广告收入 ② 一层住户甲对三四层楼板不享有共有权③ 甲出卖其住宅，乙丙丁有优先购买权 ④ 丁经乙丙同意可在楼顶建花圃⑤ 乙可分享外墙共有收益 ⑥ 丙可拒绝承担外墙维修费用
✅ ①②④⑤正确
①外墙广告收入属共有收益→全体共享 ✓
②楼板属该层层主专有部分 ✓
③业主出卖住宅，其他业主无优先购买权 ✗
④经其他共有人同意可使用共有部分 ✓
⑤共有权使乙可分享收益 ✓
⑥收益权和维修义务不可分割 ✗

🏘️ 8. 相邻关系 ★★★★

类型法条核心
用水、排水第290条尊重自然流向；人工排水需保护措施
通行第291条提供必要便利；选损失最小路线
施工、管线铺设第292条提供便利+赔偿损失
通风、采光、日照第293条不得妨碍相邻方
环境保护第294条不得排放污染/噪声/光污染
安全（防险）第295条挖掘/建造不得危及相邻不动产
📝 例题6：修房堆放建材案
甲、乙、丙比邻而居。甲修房想在乙院内堆放建材，乙不同意。甲转向丙，丙要求付费200元+不超过20天，甲同意。修房时甲搬运建材须从乙门前经过，乙阻拦。
问：下列哪一说法不正确？A.乙无权拒绝甲在其院内堆放建材 B.乙无权阻拦甲经其门前搬运建材 C.甲应依约向丙付占地费 D.超20天丙有权要求清理
✅ A（不正确）
解析：A ✗ 堆放建材不属"必须利用相邻不动产"的情形，乙有权拒绝；B ✓ 通行属必要的相邻关系便利；C ✓ D ✓ 均为正确。

📝 例题7：违反平等原则的判断
下列哪一事项违反平等原则？
A.甲(25岁)可结婚，乙(13岁)不可   B.证券公司可从事证券业务，房地产公司不可C.税务机关征税使用强制手段   D.某市合同管理干部招标中偏袒领导亲戚
✅ D
A、B是法律对主体行为能力的合理区分；C是行政关系非民事关系；D在民事活动中给予特权→违反平等原则。
📝 例题8：违反自愿原则的判断
下列哪一项违反自愿原则？
A. 赵某询问衣服价格后被摊主强卖   B. 钱某与孙某达成质押合同后孙某拒承担法律后果C. 李某被要求在接受格式合同后才能装电话   D. 周某花10元从小学生吴某手中购盗版书
✅ A
A是强迫交易→直接违反自愿原则；B是违约问题；C格式合同本身不一定违反自愿；D涉及行为能力而非自愿。

🔗 11. 民事法律关系 ★★★★★

三要素
主体（自然人/法人/非法人组织/国家）→ 客体（物/行为/智力成果/人身利益/权利）→ 内容（民事权利+民事义务）
⚠️ 不属于民法调整
情谊行为（请客吃饭、帮忙找东西、恋爱关系）——但重大过失造成损害除外｜行政关系、税收关系
📝 例题9：民法调整对象判断
下列哪些关系属于民法调整？
①税务机关退还多缴的个人所得税 ②甲发布寻物启事"拾得归还者当面感谢并付酬金" ③乙与男明星恋爱后结婚 ④丙作为青年志愿者定期去福利院帮工
✅ ②
①税务关系→行政法调整；②悬赏广告→合同关系→民法调整；③属情感行为；④志愿者服务非民事法律关系。核心判断标准：是否为平等主体之间的人身关系或财产关系。

🧑 12. 民事行为能力 ★★★★★

类型条件行为效力
完全行为能力①18岁以上 ②16岁+以自己劳动收入为主要生活来源（视为完全）可独立实施民事法律行为
限制行为能力①8岁以上未成年人 ②不能完全辨认自己行为的成年人纯获益或相适应行为→有效；其他→须法定代理人同意/追认
无行为能力①不满8岁 ②完全不能辨认自己行为的成年人一律由法定代理人代理
📝 例题10：肖特16岁音乐天才案
肖特有音乐天赋，16岁便不再上学，以演出收入为主要生活来源。其成长过程中：7岁受赠口琴1个，9岁受赠钢琴1架，15岁受赠名贵小提琴1把。判断：
A.肖特尚不具备完全民事行为能力B.受赠口琴的行为无效C.受赠钢琴有效，因与当时年龄智力相适应D.受赠小提琴无效
✅ C
肖特16岁以自己劳动收入为主要生活来源→视为完全行为能力人，A错误。但按行为发生时年龄判断：7岁（无行为能力）→受赠口琴无效（B错误）；9岁（限制行为能力）→受赠钢琴属纯获益→有效（C正确）；15岁（限制行为能力）→受赠小提琴属纯获益→有效（D错误）。

🤲 13. 监护制度 ★★★★

类型说明
法定监护未成年人：父母→祖父母外祖父母→兄姐→其他；成年人：配偶→父母子女→其他近亲属
遗嘱监护父母可通过遗嘱为未成年人子女指定监护人（第29条）
协议监护有监护资格的人之间可协议确定（第30条）
意定监护完全民事行为能力人可书面确定将来的监护人（第33条）
指定监护有争议时由居/村委会或民政部门指定，不服可向法院申请（第31条）
📝 例题11：离婚后子女监护
陆某因诈骗被判入狱，与配偶协议离婚，约定12岁孩子随陆某生活。调查发现孩子实际一直随陆某父母生活。
A.陆某是唯一监护人 B.陆某配偶是唯一监护人 C.夫妻二人均为监护人 D.陆某父母为监护人
✅ C
离婚不影响父母双方的监护人资格！虽然孩子随一方生活，但双方均为法定监护人。

📢 14. 宣告失踪 vs 宣告死亡 ★★★★

对比项宣告失踪宣告死亡
条件下落不明满 2年一般：4年；意外事故：2年（有证明不可能生存则无限制）
法律效果设定财产代管人婚姻关系消灭、继承开始、子女可被收养
撤销后代管人移交财产婚姻关系恢复（再婚除外）；财产返还

🏛️ 15. 法人分类 ★★★★

类型举例
营利法人有限责任公司、股份有限公司、其他企业法人
非营利法人事业单位法人、社会团体法人、捐助法人（基金会、宗教场所）
特别法人机关法人、农村集体经济组织法人、城镇农村合作经济组织法人、基层群众性自治组织法人（居委会/村委会）
📝 例题12：法定代表人越权签约案
甲、乙、丙成立德利实业有限责任公司，甲为董事长（法定代表人）。章程规定动用超10万元资金须经董事会同意。甲未经乙丙协商即以公司名义与丁公司签订20万元采购合同。
问：该采购合同效力如何？为什么？
✅ 合同有效
法人章程或权力机构对法定代表人代表权的限制不得对抗善意相对人（丁公司不知道也不应当知道该限制）。法人对法定代表人执行职务造成的后果承担责任。

⚡ 16. 诉讼时效 + 民事权利分类 ★★★

⏰ 一般诉讼时效：3年
自权利人知道或应当知道权利受损及义务人之日起算。法院不得主动适用诉讼时效规定。
📝 例题13：诉讼时效——法务小张签承诺书案
乙公司承建甲公司办公楼，验收合格后甲公司一直未付工程款，乙也未催要。诉讼时效届满后，乙公司组织人员到甲公司讨要，甲公司新录用法务小张以公司名义签署了"同意付款"和"承诺分期付款"的承诺书。
A.甲公司可以主张诉讼时效抗辩 B.因乙公司提起诉讼，诉讼时效中断 C.法院不应主动适用诉讼时效 D.甲公司同意履行后不能再主张时效抗辩
✅ D
诉讼时效届满后，义务人同意履行的，不得再以诉讼时效届满为由抗辩。新录用员工以公司名义签承诺书→表见代理→公司承受后果。法院不主动适用诉讼时效（C本身正确但非最佳选项）。
