## 总则编思考题

### 1. 简述民法的调整对象

民法调整**平等主体**之间的**人身关系**和**财产关系**。

- 平等主体 = 自然人、法人、非法人组织，谁都不比谁高一头
- 人身关系 = 婚姻、收养、姓名权、名誉权等
- 财产关系 = 买卖、租赁、借贷、继承等

---

### 2. 简述平等原则

民法典第4条：**民事主体在民事活动中法律地位一律平等。**

- **资格平等** — 无论国企私企、富人穷人，主体资格一样
- **地位平等** — 任何一方不得将自己的意志强加给对方
- **受保护平等** — 不因身份、所有制不同而差别对待

> 合理差别（婚龄男女不同等）不违反平等原则。

---

### 3. 简述宣告死亡的条件

民法典第46条，三个条件：

| 要件 | 内容 |
|------|------|
| **主体** | 须由利害关系人向**法院**申请 |
| **实体** | ①下落不明满**4年** / ②意外事故满**2年** / ③意外事故+证明不可能生存→不受时限 |
| **程序** | 公告（普通1年 / 意外+不可能生存3个月）→ 期满判决 |

> 宣告失踪只需2年，门槛更低。

---

### 4. 简述合伙的法律特征

- **共同出资** — 钱/实物/劳务/技术均可
- **共同经营** — 合伙人共同参与
- **共享收益** — 按约定或均分
- **共担风险** — 对外**无限连带责任**

---

### 5. 简述表见代理的构成要件

民法典第172条：无权代理但因**外观像真的**而有效。

| 要件 | 内容 |
|------|------|
| ① 无权 | 行为人实际没有代理权 |
| ② 有外观 | 存在使相对人相信有代理权的表象 |
| ③ 善意且无重大过失 | 相对人不知情且已合理审查 |
| ④ 可归责于被代理人 | 权利外观的形成与被代理人有关 |

---

### 6. 简述诉讼时效中断与中止的区别

| 对比 | 中断（195条） | 中止（194条） |
|:--:|------|------|
| **意思** | 计时器**归零重来** | 计时器**暂停保存** |
| **发生事由** | 人主动做的（起诉/主张/承认） | 客观障碍（地震/重病/被控制） |
| **发生时间** | 任何时点均可 | **最后6个月内**才适用 |
| **效果** | 已过时间作废，重新起算 | 障碍消除后继续+满6个月 |

> 口诀：中断归零重启，中止暂停续跑。

## 物权编思考题

### 1. 物权和债权的区分

| 对比 | 物权 | 债权 |
|:--:|------|------|
| **对象** | 管**物** | 管**人** |
| **特性** | 支配权 | 请求权 |
| **排他性** | 一物一权 | 一物可多债 |
| **优先力** | 物权优先于债权 | 债权平等，按比例 |
| **设立** | **法定** | **约定** |
| **追及力** | 有（追着东西走） | 无（只能找债务人） |

---

### 2. 简述物权法定原则

民法典第116条：**物权的种类和内容，由法律规定。**

| 层次 | 含义 |
|------|------|
| **种类法定** | 不能自创物权类型 |
| **内容法定** | 物权的内容不能自行约定改变 |

> 违反后果：不成立物权，最多是债权效力。

---

### 3. 善意取得的构成要件

民法典第311条，四个要件缺一不可：

| 要件 | 内容 |
|------|------|
| ① 无权处分 | 出卖人没有处分权 |
| ② 善意 | 买受人不知道+不应当知道无权处分 |
| ③ 合理价格 | 市场价成交，非无偿或明显低价 |
| ④ 公示完成 | 不动产→已登记 / 动产→已交付 |

---

### 4. 按份共有与共同共有的区别

| 对比 | 按份共有 | 共同共有 |
|------|:--:|:--:|
| **关系基础** | 无需特殊关系 | 须有特殊关系（夫妻/家庭） |
| **份额** | 有明确份额 | 无份额意识 |
| **处分份额** | 可自由转让 | 不能单独转让 |
| **分割** | 随时可请求 | 存续期间不得分割 |
| **重大处分** | 2/3以上份额同意 | 全体一致同意 |
| **外债** | 按份额承担 | 连带承担 |

---

### 5. 地役权与相邻关系的异同

**相同点：** 都处理相邻不动产利用关系。

| 对比 | 相邻关系 | 地役权 |
|------|:--:|:--:|
| **性质** | 法定限制 | 独立物权 |
| **设立** | 无需约定 | 须签合同 |
| **有偿性** | 无偿 | 通常有偿 |
| **登记** | 不需登记 | 登记对抗 |
| **程度** | 基本需求 | 增值便利 |

---

### 6. 抵押权的效力

**所担保范围（389条）：** 主债权 + 利息 + 违约金 + 损害赔偿金 + 实现费用

**对标的物效力：** 及于从物、孳息（扣押后）、代位物；不及于添附物

**对当事人效力：**

| 抵押权人 | 抵押人 |
|------|------|
| 优先受偿权（核心） | 可继续使用 |
| 保全权 | 转让需通知 |
| 可转让抵押权 | 可再次抵押（余额） |
| 实现权 | |

---

### 7. 最高额抵押权的确定事由

民法典第423条：

| # | 事由 |
|:--:|------|
| 1 | 约定期间届满 |
| 2 | 无约定满2年 |
| 3 | 新债权不可能发生 |
| 4 | 抵押物被查封扣押 |
| 5 | 债务人/抵押人破产或解散 |
| 6 | 法律规定的其他情形 |

> 口诀：到期/满两年/不可能/被查封/人没了。

---

### 8. 留置权的成立条件

民法典第447-448条：

| 条件 | 内容 |
|------|------|
| ① 合法占有 | 已合法占有债务人动产 |
| ② 同一法律关系 | 债权与留置物属同一法律关系 |
| ③ 债务到期 | 已到清偿期 |
| ④ 无相反约定 | 未约定不得留置 |

> 例外：商事留置不要求同一法律关系。

---

### 9. 区分善意占有与恶意占有的意义

| 对比 | 善意占有 | 恶意占有 |
|------|:--:|:--:|
| **心态** | 不知无权 | 明知无权 |
| **孳息返还** | 仅现存孳息 | 全部孳息 |
| **费用偿还** | 可请求必要费用 | 受限 |
| **损害赔偿** | 免责 | 负赔偿责任 |
| **时效取得** | 可取得 | 不能 |

> 核心意义：保护善意者，惩罚明知故犯者，平衡权利人利益。
